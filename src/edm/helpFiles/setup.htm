EDM Setup

When started with the "-h" option for help, edm will display the available command-line options and environment variables.

Environment Variables

  • LD_LIBRARY_PATH
    edm uses shared libraries. On most Unix operating systems, $LD_LIBRARY_PATH is used as a search path to locate those.
  • EDMFILES
    Location of fonts.list and colors.list.
  • EDMOBJECTS
    Location of edmObjects, the edm component object file.
  • EDMPVOBJECTS
    Location of edmPvObjects, the edm pv component object file
  • EDMDATAFILES
    Location of display files given as a search path similar to the PATH variable. Example: 
    .:/sns/edl/share:/sns/linac/share
  • EDMUSERLIB
    Location of user shared library file
  • EDMHELPFILES
    Location of the help files. helpMain.edl in there is opened.
